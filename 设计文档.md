# UG编程智能体 - 设计文档

## 一、整体架构（模块化）

```
UG编程助手（集成星空外挂+燕秀外挂核心功能）
├── 模块1: 刀具参数库 (ToolDB)         ── 快捷键 Ctrl+1
│   └── ★ 星空: 界面化刀库管理/夹持库/刀柄库/火花位/创建刀具实体
├── 模块2: 加工编程模板 (MachTemplate) ── 快捷键 Ctrl+2
│   └── ★ 星空: 钢料编程/加工助手/孔自动编程/粗精规则
├── 模块3: 智能推荐 (SmartRec)         ── 快捷键 Ctrl+3
│   └── ★ 星空: 编程助手/预测面/自动关联拾取
├── 模块4: 程序单生成 (ProgSheet)       ── 快捷键 Ctrl+4
│   └── ★ 星空: 后处理程式单/批量后处理/排刀报告/刀长报告
├── 模块5: AI编程顾问 (AI Advisor)     ── 快捷键 Ctrl+5
├── 模块6: 加工坐标设定 (CoordSetup)   ── 快捷键 Ctrl+6
│   └── ★ 星空: 多工件坐标/创建加工坐标系/基准点
├── 模块7: 系统设置 (Settings)         ── 快捷键 Ctrl+7
│   └── ★ 星空: 刀库参数/后处理配置/粗精规则/企业对接
├── 模块8: 辅助工具集 (ToolKit)        ── 快捷键 Ctrl+8
│   ├── ★ 星空: 刀路转曲线/刀轨画线/测量/批量改名/超级着色/刻字
│   └── ★ 燕秀: 标准件库/BOM/出图/模具资料
├── 模块9: 拆电极工具 (EDM)            ── 快捷键 Ctrl+9
│   └── ★ 星空: 拆电极/电极信息管理/基准台/EDM出图/放电图
└── 模块10: NC智能编程 (NCQuick)       ── 快捷键 Ctrl+0
    ├── ★ 核心流程: NC数据采集→智能方案生成→用户选择→一键执行
    ├── NC数据采集: 调用UG NC助手获取图档数据(尺寸/特征/材料/表面要求)
    ├── 智能方案生成: 基于采集数据自动生成多套加工方案(时间优先/质量优先/稳妥保守)
    ├── 方案对比: 并排展示时间/成本/质量/刀具磨损
    └── 一键执行: 选方案后自动创建刀具/套模板/设参数/出程序单/配坐标
```

**集中设置核心原则：**
1. 所有可配置参数一个入口，仿星空系统参数设置
2. 左侧分类树导航，右侧即时配置，改完即生效
3. 配置可导出/导入JSON，方便多台电脑同步
4. 恢复默认一键重置
5. 各模块运行时读取集中配置，不再各自独立存设置
6. 敏感参数（API Key等）加密存储

### UG版本兼容
- **基准版本**: UG NX 12.0 界面风格和API接口
- **支持版本**: NX 12.0 / NX 1847 / NX 1872 / NX 1899 / NX 1953 / NX 2007 及以上
- **界面适配**: 参照NX12.0+的Ribbon功能区风格，深色主题

### CNC后处理接口（用户自定义接入）
```
后处理接口规范：
├── post-processor/
│   ├── post-api.js          # 后处理标准API定义
│   ├── post-config.json     # 用户后处理配置
│   ├── built-in/            # 内置后处理器
│   │   ├── fanuc.js         # FANUC后处理
│   │   ├── siemens.js       # SINUMERIK后处理

│       └── template.js      # 后处理器模板
```

**用户自定义后处理器规范：**
- 遵循标准API接口，导出固定的处理函数
- 输入: 标准化加工参数JSON（刀具、转速、进给、坐标、刀轨点等）
- 输出: NC代码文本（用户自行对接机床）
- 用户只需在 custom/ 目录放入.js文件，系统自动识别加载

**预留NC代码输出字段：**

#### ⚡一刀流方案（内置，12种）
```
一刀流核心理念：减少/消除层间跳刀和区域间抬刀，实现连续高效切削。
以下方案按适用场景分类，覆盖开粗、清角、精加工全流程：

═══ 清角类一刀流 ═══

方案01: 等高清角一刀流
├── 适用: 大刀开粗后的角落清角，减少层间跳刀
├── UG操作: 深度轮廓加工(ZLEVEL_CORNER)
├── 关键参数:
│   ├── 切削方向: 顺铣
│   ├── 切削顺序: 深度优先
│   ├── 层到层: 使用转移方法
│   ├── 参考刀具: 上一把刀直径（≥上把刀直径）
│   └── 光顺: 勾选"替代为光顺连接"
├── 注意: 参考刀具直径应≥上把开粗刀直径，清角余量≥开粗余量
└── 效果: 进退刀光顺过渡，层间无跳刀

方案02: 清根参考刀具一刀流
├── 适用: 圆角/凹角区域精清根，双接触点连续切削
├── UG操作: 固定轴轮廓铣-清根驱动(FLOWCUT_REF_TOOL)
├── 关键参数:
│   ├── 驱动方法: 清根
│   ├── 参考刀具: 上一把精加工刀具直径
│   ├── 清根类型: 单路/多路/参考刀具
│   ├── 切削方向: 顺铣
│   ├── 非切削移动: 光顺进退刀，圆弧连接
│   └── 步距: 残余高度控制 ≤0.005mm
├── 注意: 清根刀半径≤参考刀具半径的1/2时用跟随工件；>1/2时用配置文件
└── 效果: 凹角区域连续清根，无抬刀空走

方案03: IPW二次开粗一刀流
├── 适用: 大刀开粗后小刀二次开粗，基于IPW残留毛坯

2. 从刀具库/模板/智能推荐导入参数 → 自动填充到程序单数据源 → 两种格式都拿到最新值
3. 刀长计算结果（最短刀长/装刀长/干涉警告）→ 工程图和电子表格同步显示
4. 余量值修改 → 工程图工序表和电子表格Sheet1同步
5. 加工时间重新计算 → 两种格式同步更新
6. 坐标系变更 → 工程图视图标注和电子表格Sheet3同步

- 辅助时间 = 换刀(8s/次) + 对刀(2min/次)
- 各工序时间明细 + 换刀次数 + 总时间
- 单件成本估算（可配工时费率）

2. **按刀具合并** — 同一把刀的多个工序合并为行组
```

├────┼────┼────┼────┼────┼────┼────┼────┼────┼────┼────┼────┤

## 十、模块10: NC智能编程 (NCQuick) ── Ctrl+0

### 核心理念
在编程任务开始前，调用UG NX的NC助手快速获取图档的多项数据，基于数据自动生成多套加工方案供用户选择，选定后一键执行快速编程。这是整个系统的"智能大脑"，将数据采集→方案生成→自动编程串联为完整闭环。

### 工作流程
```

│ 1.NC数据采集  │───▶│ 2.智能方案生成 │───▶│ 3.用户选择方案 │───▶│ 4.一键执行编程 │
```

### 步骤1: NC数据采集

**UG NC助手接口数据：**
- 零件基本信息：名称/编号/版本/单位
- 几何数据：
  - 包围盒尺寸(X/Y/Z)
  - 体积/表面积
  - 最小圆角半径
  - 最大深度/高度
  - 壁厚分析(最薄/最厚)
- 特征识别：
  - 型腔数量/深度/尺寸
  - 凸台数量/高度
  - 孔位数量/直径/深度/类型(通孔/盲孔/螺纹孔)
  - 圆角数量/半径范围
  - 倒角数量/尺寸
  - 曲面数量/面积占比
- 材料信息：
  - 材料名称/牌号
  - 硬度(HRC)
  - 加工性能评级(易/中/难)
- 表面要求：
  - 粗糙度要求(Ra)
  - 关键面标记
  - 配合面标记

**采集方式：**
- 自动采集（调用UG NX Open API / NXOpen Python脚本）
- 手动输入（Web界面表单，适用于离线场景）
- 历史记录导入（从之前保存的图档数据加载）

### 步骤2: 智能方案生成

基于采集数据，自动生成3-5套加工方案，每套方案包含：

**方案类型：**
- 方案A：时间优先 — 大刀开粗+高效半精+快速精光，最短加工时间
- 方案B：质量优先 — 高速加工策略，小切深快进给，最优表面质量
- 方案C：成本优先 — 最少换刀次数，通用刀具，最低刀具成本
- 方案D：稳妥保守 — 传统分层，留余量多，适合新手/难加工材料
- 方案E：混合策略 — 关键面精加工+非关键面高效加工

**每套方案内容：**
1. 工序列表（序号/工序名/刀具/切削参数/余量/预估时间）
2. 刀具清单（刀具类型/直径/刃数/总长/数量）
3. 预估统计：
   - 总加工时间（切削+空行程+换刀）
   - 刀具成本估算
   - 换刀次数
   - 材料去除率(MRR)
4. 风险提示（深腔刀长不足/薄壁变形/刀具干涉等）

**方案生成算法：**
- 特征→工序映射规则（型腔→开粗+清角，孔→钻孔+攻牙，曲面→半精+精光）
- 材料+硬度→切削参数推荐（引用刀具库和智能推荐模块数据）
- 几何尺寸→刀具选型（最小圆角→最大刀具直径，深度→最短刀长）
- 一刀流方案匹配（从模块2的12种一刀流方案中匹配最佳）

### 步骤3: 方案确认与微调

**方案对比视图：**
- 并排卡片展示所有方案
- 关键指标柱状图：时间/成本/质量评分
- 点击展开详细工序列表
- 标注各方案的优缺点

**微调功能：**
- 拖拽调整工序顺序
- 替换某个工序的刀具
- 修改切削参数
- 添加/删除工序
- 微调后实时更新时间和成本估算

### 步骤4: 一键执行

确认方案后，自动执行以下操作：
1. 调用模块1(刀具库)：创建所需刀具，设置参数
2. 调用模块2(加工模板)：套用对应的一刀流/加工模板
3. 调用模块3(智能推荐)：应用推荐切削参数
4. 调用模块6(坐标设定)：自动创建加工坐标系
5. 调用模块4(程序单)：生成程序单(工程图+电子表格)
6. 输出执行报告

**执行进度面板：**
- 每个步骤的执行状态（等待/执行中/完成/失败）
- 进度条
- 执行日志
- 失败时提示原因和修复建议

### 数据流
```
NC数据采集 → PartAnalysis对象 → 方案引擎 → Plan[] → 用户选择 → ExecutePlan → 各模块API调用
```

### 与其他模块的交互
- 读取模块1(刀具库)：查询可用刀具、创建新刀具
- 读取模块2(加工模板)：匹配一刀流方案、应用模板参数
- 读取模块3(智能推荐)：获取切削参数推荐
- 写入模块4(程序单)：生成程序单
- 写入模块6(坐标设定)：创建坐标系
- 读取模块7(设置)：获取编程参数默认值
- 通知模块5(AI顾问)：方案生成后AI可补充建议
